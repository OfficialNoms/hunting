[INSTALLATION]

1) Download/copy files/open in desktop etc.

2) Add to your server.cfg :
``ensure PolyZone``

``ensure bt-target``
``ensure hunting``

3) Import tiny SQL file, perhaps find pictures for your inventory hud etc.
``import items.sql``
